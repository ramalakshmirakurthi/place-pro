# **App Name**: PathPrep AI

## Core Features:

- Secure User Authentication: A robust login and registration system with password hashing (bcrypt) and JWT-based authentication for secure session management. Users are redirected to the dashboard upon successful login.
- Interactive Dashboard & Navigation: A responsive, modern UI dashboard featuring a left sidebar for navigation and a main content area. Dynamically loads module-specific content when a user clicks a sidebar item.
- Placement Test Module: Offers categorized CRT (Campus Recruitment Training) tests with multiple-choice questions, a timer, instant score calculation, and basic performance tracking. Test results are stored for review.
- Core Engineering Video Player: A section featuring categorized core engineering videos with an embedded player. Includes search functionality, filter by branch, and allows users to mark videos as watched.
- AI Notes Generation Tool: An AI-powered system that generates structured, placement-focused study notes from a user-provided topic (e.g., 'Stack Data Structure') using the OpenAI API. Notes include definitions, key concepts, examples, code snippets, and interview questions.
- AI Study Resource Tools: Provides options to generate quizzes directly from the AI-generated notes or download the notes as a PDF for offline study.
- Personalized Performance Analytics: Displays individual user performance history, including test scores and identification of weak topics from CRT tests, to help users focus their study efforts.

## Style Guidelines:

- Light color scheme to enhance readability for study materials.
- Primary color: Deep intellectual purple (#512999) to signify wisdom and knowledge.
- Background color: A subtle, desaturated light grey-purple (#F4F1F7) for a clean, focused environment.
- Accent color: A vibrant, energetic blue (#194CFF) for call-to-action buttons and interactive elements, providing a modern contrast.
- Headlines and prominent text: 'Space Grotesk' (sans-serif) for a modern, tech-savvy and scientific feel.
- Body text and general UI: 'Inter' (sans-serif) for excellent readability and a neutral, objective aesthetic across all content.
- Code snippets: 'Source Code Pro' (monospace sans-serif) for clear and legible display of programming examples.
- Utilize clean, modern vector icons (linear or filled) that represent learning, technology, progress, and engineering concepts, ensuring consistency and scalability.
- Responsive layout featuring a distinct left sidebar for navigation and a spacious main content area, optimized for readability and user interaction across various screen sizes.
- Subtle and smooth transitions for content loading and state changes, ensuring a fluid and professional user experience without distraction.
